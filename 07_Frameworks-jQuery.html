Why do or did people use jQuery

Unified access via $ function

Honestly it's shorter to write $ instead of document.querySelectorAll, or addEventListener. Also things are in one spot.

Makes selecting elements easier

Before querySelectorAll it provided the same functionality $('div') $('div:nth-of-type(5)')

Easy AJAX

$.ajax({url:"...",success:data=>{}}) $.get(url,onsuccess()) $.post(url,{data},onsuccess())

Simple DOM manipulation and creation

$('div').remove() // removes all divs $('.some-div').html(`Hello!`)

Simple event listener handling

let handler = event => console.log(event); $('div').on ( 'click', handler ) $('div').off( 'click', handler ) $('div').off( 'click' )
