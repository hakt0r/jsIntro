Summary

Creating Elements

        const ball = document.createElement('div');
        // document.write() // don't use it

Setting CSS classes

        ball.classList.add('cssBall');
        ball.classList.remove('cssBall');
        ball.classList.toggle('cssBall');
        ball.classList.value = 'class1 class2 class3';

Inserting elements /append/prepend/...

            document.body.append(ball)
        //           .prepend(ball)
        //       ball.appendTo(document.body);
        // document.body.insertAdjacentElement( position, ball );
        //   => the broadsword of element insertion

Getting information about the window

        window.innerHeight // read only: height of the window
        window.innerWidth  // read only: width  of the window

Do timed loops or wait to execute code

        setInterval ( callback, timeInMS  );
        setTimeout  ( callback, timeInMS  );
        // clearTimeout, clearInterval // mdn

Select existing elements from the document

        document.querySelector()        // one element with css selector
        document.querySelectorAll()     // multiple by class
        document.getElementById()       // one element by id ( no # )
        document.getElementsByClassName // multiple by class ( no . )
        // ...
        //   const $ = document.querySelectorAll
        //   $('.ball')

Camelization Rule

        ball.style['font-size'] = '12px';
        ball.style.fontSize     = '12px';

Real CSS

        // translate into CSSOM
        ball.style.cssText = `
            width:  100px;
            height: 100px;
            border-radius: 50px;
            background-color: green;
            font-weight: bolder;
        `;
        // translate into CSS
        console.log(
            ball.style.cssText
        );
